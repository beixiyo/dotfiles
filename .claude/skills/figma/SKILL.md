name: figma
description: 仅当用户提供 Figma 链接、Figma MCP 数据，或明确要求“按 Figma 设计稿实现/还原”时使用。不处理普通 UI 实现
---

## 规则
- 先读取 Figma 节点结构、尺寸、颜色、字体、间距和资源
- 优先还原设计稿，不自行改设计风格
- 缺失资源、字体或交互状态时明确说明，不伪造像素级还原
- 代码规范交给 `react` / `vue`、`ui-design`、`layout`、`responsive`

## 输出
- 标出设计稿来源节点
- 标出无法还原或需要用户补资源的部分
- 实现时遵循当前项目已有技术栈和组件规范
